使用说明
准备数据文件
将你的旅游数据表命名为 tourism_data.csv，放在与脚本相同的文件夹下。

安装依赖（如果还没有安装）
打开终端/命令提示符，运行：

bash
pip install pandas numpy
运行脚本
在终端执行：

bash
python clean_tourism.py
查看结果

清洗后的数据：cleaned_tourism_data.csv

控制台会打印详细的清洗报告，包括删除的行数、各列统计信息。

代码设计特点（适合初学者）
逐步处理：先删重复，再补缺失，再转格式，最后再删一次重复。逻辑清晰。

避免复杂函数：没有自定义复杂的类，只用 pandas 内置方法。

异常值处理：年龄超出合理范围、消费金额负数等都被修正。

打印信息丰富：每一步都有输出，方便你观察数据变化。

容错性好：使用 if "列名" in df.columns 检查列是否存在，避免报错。

你可能需要微调的地方
文件编码：如果你的 CSV 不是 UTF-8 编码，可以把 encoding="utf-8" 改成 "gbk" 或 "gb2312"。

列名：请确保代码中的中文列名（如 "游客年龄"）和你表格里的完全一致，否则需要修改。

如果你在运行时遇到任何错误，可以把报错信息发给我，我帮你调整。
